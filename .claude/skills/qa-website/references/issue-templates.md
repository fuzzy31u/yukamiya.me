{timestamp}

## 詳細

### 件数比較
| ソース | 件数 |
|--------|------|
| about-content.js | {js_count} |
| about.md | {md_count} |
| 差分 | {diff} |

### 欠損アイテム
{missing_items}

## 推奨アクション
- [ ] 欠損アイテムを about-content.js に追加

## Issue 作成コマンド例

### コンテンツ不整合

```bash
gh issue create \
  --title "[QA] コンテンツ不整合: media セクション (5件の差分)" \
  --body "$(cat <<'EOF'
## 検出日時
2025-01-01 12:00:00 JST

## 概要
`media` セクションで about-content.js と about.md の間に 5 件の不整合が検出されました。

## 詳細

### 件数比較
| ソース | 件数 |
|--------|------|
| about-content.js | 6 |
| about.md | 11 |
| 差分 | -5 |

### 欠損アイテム
- YOUTRUST プロダクトヒストリーカンファレンス (2025/01)
- デジタルプラクティス論文発表 (2025/01)
- Women Tech Terrace 2024 (2024/08)
- Women Tech Terrace 2024 Report (2024/07)
- Waffle & IT ジェンダーギャップ (2024/03)

## 推奨アクション
- [ ] 欠損アイテムを about-content.js に追加
- [ ] 各 URL が有効か確認

---
Generated by yukamiya.me QA Skill
EOF
)" \
  --label "qa,content,automated"
```

### リンク切れ

```bash
gh issue create \
  --title "[QA] リンク切れ検出: example.com/page" \
  --body "$(cat <<'EOF'
## 検出日時
2025-01-01 12:00:00 JST

## 概要
以下の URL でリンク切れが検出されました。

## 詳細

| URL | ステータス | 場所 |
|-----|-----------|------|
| https://example.com/page | 404 | about-content.js (media) |

## 推奨アクション
- [ ] URL が正しいか確認
- [ ] 代替 URL があれば更新

---
Generated by yukamiya.me QA Skill
EOF
)" \
  --label "qa,broken-link,automated"
```

## ラベル作成コマンド

```bash
gh label create qa --description "QAスキルで検出された問題" --color 1d76db
gh label create content --description "コンテンツ関連の問題" --color 0e8a16
gh label create broken-link --description "リンク切れ" --color d73a4a
gh label create ui-ux --description "UI/UX関連の問題" --color f9a825
gh label create automated --description "自動生成されたIssue" --color bfdadc
```
